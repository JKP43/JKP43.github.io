Autonomous AI for Recycling

Python AWS YOLOv5 MLFlow

Overview

Built a computer-vision pipeline that classifies recyclable materials in real time using a custom-trained model deployed on AWS, enabling smarter automated waste sorting.

What it does

The system uses a custom-trained YOLOv5 model to identify and classify different types of recyclable materials in real-time, enabling automated waste sorting in industrial settings.

My role

I was responsible for deploying the model on AWS as part of the cloud team, and also for managing the MLFlow experiments as well as creating the automation scripts for fetching the data, training the model, and pushing it to production.

Tech stack

  • Python — core pipeline and model logic
  • AWS — cloud deployment and inference
  • YOLOv5 — object detection model
  • MLFlow — experiment tracking and model management
