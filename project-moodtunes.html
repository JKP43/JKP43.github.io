Mood Tunes

Kotlin Android REST API

Overview

An Android app that detects the user's current mood via a short quiz and curates a personalized playlist by querying a music REST API — built natively in Kotlin.

What it does

Users can choose their emotions based on the emojis on the cards, and it will fetch a recommended song from the Spotify API based on their selection.
