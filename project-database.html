Database Website

CS 340 — Introduction to Databases · Oregon State University

Python MySQL Flask

Overview

A full-stack web application built for CS 340 at Oregon State University that manages data using a relational database, supporting full CRUD operations through a browser-based interface backed by MySQL.

What it does

From a fictional company, created a database management website that tracks the manufacturing companies of the products, amount of sales made, and the customers that buy the compan'y products. It records Sales of Products to Customers and the Manufacturers of the Products.
